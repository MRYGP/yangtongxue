# Weekly Review (2025-W51)

## Data
- 运动：完成次数 / 总时长 / 触发条件（如：放学后/晚饭前）
- 学业：错题数 / 复盘次数 / 卡点类型
- 项目：产出物进度（charter/log/script 等）

## Insights
- 哪个触发条件最容易让孩子开始行动？
- 哪个任务最容易"拖延/抵触"？原因是什么？

## Adjustments
- 下周最小化目标是什么？
- 哪个任务需要降门槛或拆更小？
