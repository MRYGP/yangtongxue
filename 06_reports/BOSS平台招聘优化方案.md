# BOSS平台招聘优化方案（可执行SOP）

> 目标：提高 BOSS 曝光与投递/沟通率，让"合适的人不被吓退"，同时保留真实标准（能稳定线下、能跑闭环、愿意学AI）。

## 一、平台设置（优先级最高：先扩大曝光池）

### 1）必改项（直接影响曝光池）

| 字段 | 当前 | 建议改成 | 原因 |
|------|------|----------|------|
| **学历要求** | 硕士 | **本科及以上** 或 **不限** | "硕士"会直接过滤大量在校生/优秀本科，曝光池显著变小 |
| **经验要求** | 不详/较高 | **不限 / 在校生** | 兼职岗核心是稳定+可带教，不是"资深老师" |
| **职位名称** | 学习研究员…AI协作… | 用"家教/兼职"用户会搜的词 | 标题决定点击率；"研究员/AI协作"会劝退目标人群 |
| **薪资展示** | 200–250元/天 | **按小时/按次**（见下文推荐写法） | "按天"不清楚几小时，容易被误解/被跳过 |
| **职位类型** | 教师 | **家教/兼职老师/陪读**（平台可选项取最接近） | 提升匹配推荐准确度 |

### 2）推荐标题（做 A/B 测试：3个版本轮换）

建议一次只用一个标题跑 48–72 小时，看"曝光→沟通→面试"漏斗。

- **A（最稳）**：`小学陪读老师｜周末兼职｜雨花区`
- **B（更抓眼）**：`六年级陪读/作业辅导｜75元/小时｜提供ChatGPT PLUS`
- **C（强调差异化但不吓人）**：`学习教练（家教兼职）｜会带你学AI工具｜雨花区`

> 规则：标题里尽量包含 **年级/兼职/区域/薪资** 这4类关键词中的至少2个。

### 3）薪资推荐写法（同样的钱，表达不同=投递率不同）

你当前在 BOSS 写的"200–250/天"不算高，但对在校生会出现两种误读：
1）"一天要干很久"；2）"不确定一天几小时"→直接跳过。

更推荐写成（按你给的口径）：
- **按小时**：`75元/小时（含课后10分钟记录）`
- **按次（更直观）**：`每次到场通常1–2小时：150–180元/次；更长时段（约2.5–3小时）：200–250元/次（按强度）`

### 4）福利/亮点（建议在平台"职位亮点/福利"里写）

- 提供 **ChatGPT PLUS** 账号（不用自费）
- 有人带教 AI 工具与模板（不用一开始就会）
- 每周复盘一次，帮助你把产出整理成可展示材料（可写进简历）

## 二、职位描述正文（直接复制到BOSS）

> 建议控制在 1000–1800 字：30秒能看完，降低跳出。

```text
【一句话说明白】
不是只讲题的传统家教。
我们带一个六年级男孩做「学习+运动+小项目」的综合陪跑，你负责线下执行和简单记录。

【你要做什么】（每次课 3 步）
1）课前5分钟：和家长对齐"目标一句话"（只抓一个小主题）
2）课中60–90分钟：带孩子完成一小步，留下可验证证据（一页解题过程/一段复述/一个小作品）
3）课后10分钟：写4行事实记录（用时/做了什么/结果/下一步）
备注：不写长文、不写作文；我们用 GPTS 自动把4行记录生成"反馈卡"发给孩子看。

【硬要求只有3条】
✓ 能稳定线下到场（长沙雨花区）
✓ 每次课后10分钟能提交4行记录（有模板）
✓ 愿意用AI工具协作（我们提供ChatGPT PLUS账号 + 手把手带教）

【加分项】（不是必须）
- 师范/教育/心理/理工科背景
- 有辅导经验
- 周末时间更充裕

【薪资】（更适合兼职/在校生的写法）
75元/小时（含课后10分钟记录）
每次到场通常1–2小时：150–180元/次
更长时段（约2.5–3小时）：200–250元/次（按强度）

【时间安排】
- 平时：晚间为主，每周2–3次
- 周末：可安排更长时段
- 先合作4周，验证你能稳定跑通"上课→记录→调整"的闭环；合适再长期

【你能得到什么】
- 每周产出可展示的教学记录/方法卡（可写进简历，可脱敏）
- 学会用AI辅助教学与复盘（我们带教）
- 家长每周陪你复盘，帮你把成果整理出来

【怎么申请】（只要4行）
1. 可上课时间段与频率
2. 最擅长两门内容
3. 是否接受课后10分钟记录
4. 方便试讲时间
```

## 三、主动沟通话术（提升沟通率：别等投递）

> BOSS 上"主动打招呼"往往比被动等投递更有效。建议每天主动沟通 10–20 人，连续 3 天看变化。

### 1）第一句（短、轻、可复制）

```text
你好～我们在长沙雨花区找一位六年级陪读/作业辅导的兼职老师（线下为主）。
不要求你一开始就会AI，我们提供ChatGPT PLUS账号并带教；你只要能稳定到场、课后10分钟写4行记录即可。
方便发下你可上课时间 + 擅长科目吗？
```

### 2）对方问"我不会AI怎么办"

```text
没关系，我们会教，而且会给你配ChatGPT PLUS账号。
你只需要把课后4行事实记录写出来，AI会帮你生成反馈卡/备课框架，你按模板跑就行。
```

### 3）对方问"是不是要立刻提分"

```text
我们不追求"马上神效"，但每次课都要有可见产出（证据）。
先合作4周，能看到趋势并持续调整；如果节奏不合适我们也会及时止损。
```

## 四、筛选问题（减少无效沟通）

建议在聊天里用"4行申请"快速筛掉不稳定的人（你仓库里也有 `招聘应聘者速览.md` 可直接发对方）。

### 4行申请（对方复制填空）

```text
可上课时间段与频率：
最擅长两门内容：
是否接受课后10分钟记录：
方便试讲时间：
```

### 一票否决项（提前说清）

- 不能稳定线下到场
- 抵触课后10分钟记录
