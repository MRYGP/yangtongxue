# 招聘JD：学习教练 / 家庭教育陪跑（线下，AI协作）

地点：长沙雨花区（线下为主）
对象：11岁｜六年级男孩（只带这一个孩子）

一句话：不是“只讲题”的传统家教，我们做的是「AI + 人」的学习闭环。
补课大约占30%，更重要的是把学习变成可复盘的小训练：目标更小、证据更清楚、每次课都能推进一小步。

## 这份工作适合谁（降低门槛版）

- 你不需要“教育专家”，也不需要会写很复杂的 prompt
- 你只要愿意按模板跑：上课 → 4行记录 → GPTS自动生成反馈卡
- 你擅长现场：建立信任、观察状态、带孩子完成一小步

## 你要做的三件事（每次课）

1) 课前5分钟：对齐“目标一句话”
和家长确认：本次课只抓一个最小主题、孩子当前卡点、以及要留下的可见产出。

2) 课中60–90分钟：带孩子完成一小步（必须留证据）
可验证证据示例：一页解题过程、一段复述、一个小作品。

3) 课后10分钟：提交4行事实记录 + 下次一条改法
只写：用时 / 做了什么 / 结果 / 下一步一小步（30分钟内可启动）。不写长文、不写作文。

## 必须项（硬要求，只有3条）

- 能稳定线下到场（长沙雨花区）
- 课后10分钟内能提交“4行记录”（我们有模板）
- 愿意使用 ChatGPT/GPTS 协作（我们会给老师配 PLUS 账号 + 有人带教，上手很快）

## 加分项（不是必须，但有更高匹配度）

- 周末/节假日偶尔能加课或可安排更长时段（不强制）
- 教育学/心理学/师范/理工科背景，或有学习辅导经验
- 有“证据意识”：喜欢把过程写清楚、把结果留出来
- 学历：硕士加分；本科优秀也欢迎（以稳定交付为准）

## 即时反馈卡（我们为什么要求它）

每次课后会用 GPTS 自动生成“即时反馈卡”（<10分钟），包含：五维能力经验值变化 / 升级状态 / 解锁权限 / 距离里程碑 / 下次小目标。
你只负责4行记录；反馈卡由工具自动生成。
模板：`docs/模板_课后即时反馈卡.md`

## 工作量边界（避免误解）

- 每周总投入：约3–5小时（以实际上课频次为准）
- 备课：每次不超过30分钟（工具会给框架）
- 记录：每次不超过10分钟（只要4行事实）

这不是一份"上课-拿钱-走人"的兼职。如果你愿意，这是一个**把教学经验变成职业作品集**的机会。

### 短期（0-6个月）：建立你的教育作品集

**你会产出什么**：
- **每周1份可展示材料**（可脱敏）：
  - 训练方案设计（"审题训练三色笔法"）
  - 学生成果对比（"错题率从60%降到20%"）
  - 方法卡/工具卡（"作业时间管理三步法"）
  - 周总结/案例记录
  
- **每月1个深度案例**（200-500字）：
  - 问题诊断 → 方案设计 → 执行过程 → 数据验证
  - 可用于简历、面试、公众号、求职作品集
  
- **4个月后你会拥有**：
  - 4-6个完整案例（覆盖学科/习惯/身体发育/AI素养）
  - 1套自己的教学工具库（标注卡/表达模板/时间管理表等）
  - 1份数据化的教学成果报告（用折线图/对比照片/录音对比说话）

**实际例子**（脱敏）：
> "我带过一个六年级学生，数学应用题经常漏条件。我设计了'三色笔标注法'，4周后他的漏条件次数从6次/周降到2次/周。我把这个案例整理成1000字材料，面试时HR说'这个案例很具体，你有方法论'。"

### 中期（6个月-2年）：成为"研究型教育者"

**你会训练的4项通用能力**（这些能力在任何行业都值钱）：
1. **问题诊断能力**：从表面现象找到根本原因（不是"粗心"，而是"缺少审题工作流"）
2. **方案设计能力**：把想法变成可执行方案（时间线/工具/验证指标）
3. **证据意识**：用数据说话，而不是"我觉得"（错误率/完成时间/流畅度）
4. **复盘与迭代**：看数据调整方案（如果2周没起色，会调整哪个参数）

**每周1次陪跑复盘**（我会陪你）：
- 把你做的事情整理成案例（教你怎么写）
- 帮你提炼方法论（从具体经验抽象出可复用框架）
- 指导你如何脱敏、如何让案例更有说服力

**1年后你会拥有**：
- **10-15个深度案例**（可以写成教育博客、公众号、知乎）
- **1套自己的方法论体系**（不再是"我会教"，而是"我有系统"）
- **1份季度研究报告**（3000字，含背景/问题/方案/数据/反思）
- **可量化的职业证据**："我设计的方案让学生XX指标提升XX%"

**实际转化路径**：
- 转行做教育产品经理：你有真实用户研究和方案落地经验
- 转行做教育咨询：你有案例库和方法论
- 考研/出国申请：你有research经历和可展示作品
- 继续做老师：你有差异化竞争力（"研究型教育者"vs"只会讲题的家教"）

---

### 长期（2年+）：职业转型与影响力

**如果你做得好，2年后你可以**：

1. **建立个人品牌**：
   - 写教育类公众号/知乎专栏（你有20+个真实案例）
   - 做线上课程/训练营（把你的方法论教给其他老师/家长）
   - 参加教育峰会/分享（"我如何用游戏化系统让孩子爱上运动"）

2. **拓展职业选项**：
   - 教育创业：开工作室，用你验证过的方法服务更多学生
   - 教育科技公司：你懂产品（知道什么工具真正有用）+ 懂用户（知道家长和学生真实痛点）
   - 教育研究：你有研究方法和数据积累

3. **年度成果白皮书**（我会帮你整理）：
   - 5-10个深度案例（每个3000字）
   - 方法论总结（你的教学框架）
   - 工具库（你设计的所有工具）
   - 数据汇总（服务了多少学生，效果如何）
   - 可以用于：求职作品集/考研材料/出版书籍/线上课程

**真实案例参考**（其他项目）：
> "我之前带的一位研究生，用类似方法带学生2年，积累了15个案例。她后来转行做教育产品经理，面试时拿着案例集说'我有真实的用户研究和迭代经验'，3天拿到offer。"
