# 招聘JD：学习教练 / 家庭教育陪跑（线下，AI协作）

地点：长沙雨花区（线下为主）
对象：11岁｜六年级男孩（只带这一个孩子）

一句话：不是“只讲题”的传统家教，我们做的是「AI + 人」的学习闭环。
补课大约占30%，更重要的是把学习变成可复盘的小训练：目标更小、证据更清楚、每次课都能推进一小步。

## 这份工作适合谁（降低门槛版）

- 你不需要“教育专家”，也不需要会写很复杂的 prompt
- 你只要愿意按模板跑：上课 → 4行记录 → GPTS自动生成反馈卡
- 你擅长现场：建立信任、观察状态、带孩子完成一小步

## 你要做的三件事（每次课）

1) 课前5分钟：对齐“目标一句话”
和家长确认：本次课只抓一个最小主题、孩子当前卡点、以及要留下的可见产出。

2) 课中60–90分钟：带孩子完成一小步（必须留证据）
可验证证据示例：一页解题过程、一段复述、一个小作品。

3) 课后10分钟：提交4行事实记录 + 下次一条改法
只写：用时 / 做了什么 / 结果 / 下一步一小步（30分钟内可启动）。不写长文、不写作文。

## 必须项（硬要求，只有3条）

- 能稳定线下到场（长沙雨花区）
- 课后10分钟内能提交“4行记录”（我们有模板）
- 愿意使用 ChatGPT/GPTS 协作（我们会给老师配 PLUS 账号 + 有人带教，上手很快）

## 加分项（不是必须，但有更高匹配度）

- 周末/节假日偶尔能加课或可安排更长时段（不强制）
- 教育学/心理学/师范/理工科背景，或有学习辅导经验
- 有“证据意识”：喜欢把过程写清楚、把结果留出来
- 学历：硕士加分；本科优秀也欢迎（以稳定交付为准）

## 即时反馈卡（我们为什么要求它）

每次课后会用 GPTS 自动生成“即时反馈卡”（<10分钟），包含：五维能力经验值变化 / 升级状态 / 解锁权限 / 距离里程碑 / 下次小目标。
你只负责4行记录；反馈卡由工具自动生成。
模板：`docs/模板_课后即时反馈卡.md`

## 工作量边界（避免误解）

- 每周总投入：约3–5小时（以实际上课频次为准）
- 备课：每次不超过30分钟（工具会给框架）
- 记录：每次不超过10分钟（只要4行事实）

## 你能得到什么（对候选人真正有价值的部分）

- 每周至少产出1份可展示材料：训练方案/方法卡/周总结/学生成果对比（可脱敏）
- 训练4项通用能力：把目标拆小、留证据、数据复盘、把过程讲清楚写成材料
- 我每周会陪你复盘一次：把你做的东西整理成可展示经历与作品

补充说明：这份工作本质上也是一个“老师在学习与迭代”的过程。我们不要求你一上来就有神奇效果，但要求你能稳定跑闭环——每次课都有可见产出，4周能看到趋势并据此调整。

## 时间安排与合作方式（降低试错成本）

- 平时：每周3–4次，每次1–2小时（以晚间/周末为主）
- 合作方式：先合作4周，验证你能稳定跑通“上课→记录→调整”的节奏；合适再长期续约

## 薪资（强烈建议写清，否则投递会很少）

建议在平台按“兼职/在校生也能接受”的方式写清楚（同一套钱，表达不同投递率差很多）：

- 按小时：75元/小时（含课后10分钟记录）
- 按次（更推荐）：每次到场通常1–2小时，约150–180元/次；如果当天连上更长时段（约2.5–3小时），约200–250元/次（根据强度）

并注明：试讲是否付费、路费补贴、加课加价规则（避免反复沟通）。
