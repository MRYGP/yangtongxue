# 课后即时反馈卡模板

## 文件命名
`03_logs/YYYY-MM-DD-feedback.md`（例如：`03_logs/2025-12-29-feedback.md`）

## 使用说明

**触发时机**：每次课后10分钟内

**生成方式**：
1. 研究生记录课后4行（用时、做了什么、结果、下一步）
2. 粘贴到GPTS
3. GPTS自动生成即时反馈卡
4. 发给孩子看

**设计原则**：
- ✅ 反馈速度：<10分钟（符合即时反馈理论）
- ✅ 多感官刺激：数字变化、升级特效、鼓励性语言
- ✅ 进度可视化：短期+中期+长期
- ✅ 下一步明确：30分钟可启动
- ✅ 高效学习方法加成：闭书回忆、AI辅助检索、睡前复习（基于科学研究）

## 关键设计细节

### 1. 反馈速度

- ✅ **<10分钟**：课后立即生成
- ✅ 不要等到晚上或第二天

### 2. 多感官刺激

- **视觉**：
  - 数字变化（+18经验值）
  - 升级特效（🎉 升级！）
  - 进度条（████████░░）
- **情感**：
  - 鼓励性语言（"太棒了！"）
  - 里程碑庆祝（"连续5天达成！"）

### 3. 进度可视化

- **短期**：今日完成度
- **中期**：本周进度
- **长期**：距离下一个里程碑

### 4. 下一步明确

- ✅ 30分钟可启动
- ✅ 具体可执行
- ✅ 最多3条（避免overwhelm）

### 5. 鼓励为主

- ✅ 正反馈为主（做对了→升级）
- ✅ 容错空间（做错了→也有"尝试经验值"）
- ❌ 不扣分（避免焦虑）

## 常见问题

### Q1：如果本次课没有任何经验值增加怎么办？

**A1**：也要生成反馈卡，但要特别鼓励：
```
📊 五维能力变化
本次课主要是休息和调整，没有直接的经验值变化。
但休息也是进步的一部分！明天继续加油！

💪 下次小目标
1. [ ] 恢复运动打卡
2. [ ] 从简单题目开始
```

---

### Q2：如果孩子表现不好，反馈卡怎么写？

**A2**：不扣分，但明确指出下一步改进方向：
```
📊 五维能力变化
🥉 学业掌握度：+0经验值 ⚠️ 需关注
   └─ 3道题只对1道，需要再练习

💪 下次小目标
1. [ ] 从最简单的题目开始，先建立信心
2. [ ] 每道题先自己想3分钟，再问老师
```

---

### Q3：如果孩子升级了，反馈卡怎么特别庆祝？

**A3**：使用特殊格式：
```
🎉🎉🎉 恭喜升级！ 🎉🎉🎉

你的"学业掌握度"从新手升级到熟练了！

解锁新权限：
✅ 下周可以自选学习内容
✅ 可以给同学讲题

继续加油，向"专家"进发！
```

---

### Q4：高效学习方法加成如何计算？

**A4**：经验值计算示例：
```
基础经验值：攻克3道错题 = +18经验值

使用高效学习方法：
✅ 闭书回忆（回忆准确率80%）→ ×1.2倍 = 21.6经验值
✅ AI辅助检索 → 额外+8经验值
✅ 睡前复习 → 额外+5经验值

总计：21.6 + 8 + 5 = 34.6经验值（比普通学习多92%）
```

**注意**：
- 闭书回忆的×1.2倍加成只对基础经验值生效
- 额外经验值（+8、+5）是独立计算的
- 连续7天使用高效学习方法有里程碑奖励（+20经验值）

## 使用提示

1. **每次课后必须生成**：即使没有经验值变化也要生成
2. **10分钟内发给孩子**：不要拖延
3. **鼓励为主**：正反馈占80%，改进建议占20%
4. **下一步明确**：让孩子知道"我下次要做什么"
5. **进度可视化**：让孩子看到"我离目标还有多远"
