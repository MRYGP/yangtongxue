# 课程决策手册（家长版）

## 使用说明

本手册是给**家长**使用的决策指南。你的任务是**规定所有内容、流程、标准**，让老师只需要执行。

**核心原则**：
- 你负责：规定内容、流程、标准、调整方向
- 老师负责：执行流程、记录数据、现场微调（5%以内）、反馈问题
- 所有重大决策都由你做出

---

## 一、角色定位

### 你的职责

1. **课程设计**：规定90分钟流程、各环节内容、时间分配
2. **目标设定**：设定身体发育目标、学业目标、其他目标
3. **标准制定**：制定数据追踪标准、评估标准
4. **调整优化**：根据数据反馈，调整课程内容和目标
5. **质量把控**：监督老师执行情况，确保按标准执行

### 老师的职责

1. **严格执行**：按照你规定的流程执行
2. **数据记录**：准确记录所有数据
3. **问题反馈**：及时反馈执行中的问题
4. **微调执行**：在5%范围内微调（如时间分配）

### 课内/课外边界（理解90分钟的边界）

⚠️ **重要：90分钟课程不是"把所有事情都做完"**

**课内（老师在场，90分钟）**：
- 认知热身（舒尔特表）
- 身体发育检查（检查数据，不是做运动）
- 主题教学（学业/AI/科创/表达）
- 实践练习（动手做）
- 复盘总结
- 冥想引导（可选：课内或课后）

**课外（孩子自己做）**：
- **每天运动**：30-60分钟（跳绳、篮球、拉伸等）
- **每天学习**：作业、复习、错题整理
- **每天或定期冥想**：10-15分钟

### 老师的角色定位

- ✅ **教练**：检查数据、分析趋势、指导方法、设定目标、教学、记录
- ❌ **保姆**：什么都管（运动、作业、生活习惯）
- ❌ **陪练**：陪着孩子做运动、陪着做作业

**为什么这样设计**：
1. **时间限制**：90分钟装不下所有事情
2. **效率考虑**：孩子每天在家做，比课内做1次效果好
3. **培养习惯**：让孩子养成自己做的习惯，不依赖老师
4. **专业分工**：老师的价值在于"分析和指导"，不是"陪练"

**家长的角色**：
- 监督孩子课外执行（运动打卡、作业完成）
- 提供数据给老师（运动打卡表、学习记录）
- 远程检查（如果老师不在场）

## 二、如何设定身体发育目标

⚠️ **重要前提：运动在课外做，老师课内只检查数据**

### 课内 vs 课外的分工

**课外（孩子执行，家长监督）**：
- 每天运动30-60分钟（跳绳、篮球、拉伸等）
- 记录数据（运动打卡表：日期、项目、次数/时长）
- 作息营养（睡眠、牛奶鸡蛋等）

**课内（老师检查和指导）**：
- 查看运动打卡数据（3分钟）
- 分析趋势和问题（3分钟）
- 设定下周目标（4分钟）

**家长需要提供的数据**：
- 运动打卡表（纸质或电子都可以）
- 格式示例：
```
  2025-01-17（周五）
  - 跳绳：300个，用时8分钟
  - 拉伸：5分钟
  
  2025-01-18（周六）
  - 篮球：30分钟
```

**老师需要做的**：
- 检查数据，分析趋势
- 设定下周目标（基于你提供的标准）
- 记录到课后4行记录中

---

### 第一步：摸底评估（第一次课）

**运动摸底**：
- 查看过去一周的运动打卡数据
- 计算完成率、平均时长
- 识别问题和亮点

**营养摸底**：
- 询问：牛奶、鸡蛋、鱼肉、碳酸饮料、零食
- 记录现状
- 识别需要改善的点

**睡眠摸底**：
- 询问：睡觉时间、起床时间、睡眠时长、睡前习惯
- 记录现状
- 识别需要改善的点

**心理摸底**：
- 询问：整体心情、身高焦虑、学业压力、同伴关系
- 评分（1-5分）
- 识别需要关注的点

### 第二步：设定目标（基于摸底结果）

**运动目标示例**：

| 现状 | 目标 | 执行方式 |
|------|------|----------|
| 完成率71%（5/7天） | 完成率85%（6/7天） | 每天跳绳300个（分3组） |
| 平均时长18分钟 | 每次≥20分钟 | 周末增加篮球或游泳30分钟 |
| 无拉伸 | 睡前拉伸10分钟/天 | 每晚睡前必做 |

**营养目标示例**：

| 现状 | 目标 | 执行方式 |
|------|------|----------|
| 牛奶500ml/天 ✅ | 保持 | 早餐250ml + 睡前250ml |
| 碳酸饮料1-2次/周 ⚠️ | 减少到0 | 用酸奶、鲜榨果汁替代 |
| 零食（薯片糖果）多 ❌ | 改为健康零食 | 坚果、水果、酸奶 |

**睡眠目标示例**：

| 现状 | 目标 | 执行方式 |
|------|------|----------|
| 10:30入睡 ⚠️ | 10:00入睡 | 9:30开始准备（洗漱+拉伸） |
| 睡眠8小时 ❌ | 睡眠9-10小时 | 10:00睡 → 6:30-7:00起 |
| 睡前玩手机30分钟 ❌ | 睡前1小时不看屏幕 | 9:00后手机交给家长 |

**心理目标示例**：

| 现状 | 目标 | 应对策略 |
|------|------|----------|
| 身高焦虑2分（有点在意） | 降低到3-4分 | "身高是可以努力的，我们一起做运动+营养+睡眠，一定会有效果。" |
| 学业压力3分（中等） | 保持或降低 | "成绩5-10名已经很好了，不用太焦虑。" |

### 第三步：制定话术模板（给老师用）

**运动话术示例**：

"本周目标：
- 完成率提升到85%（6/7天）
- 每次运动时长≥20分钟
- 新增项目：睡前拉伸（10分钟/天）

具体安排：
- 跳绳基础量：每天300个（分3组）
- 周末增加：篮球或游泳30分钟
- 每晚睡前：拉伸10分钟（必做）"

**营养话术示例**：

"营养方面，你的牛奶和鸡蛋都做得很好！但碳酸饮料和薯片要少吃，因为它们会影响钙的吸收。我们用酸奶和坚果替代，好不好？"

**睡眠话术示例**：

"生长激素在晚上10点到凌晨2点分泌最多，所以我们要10点前睡着。这周目标是9:30开始洗漱，10:00上床，手机9点后交给爸妈，好吗？"

## 三、如何分析数据

### 每周数据分析

**身体发育数据**：

1. **运动完成率趋势**

示例数据：
```
第1周：71%（5/7天）
第2周：85%（6/7天）↑
第3周：57%（4/7天）↓
第4周：71%（5/7天）↑
```

分析：
- 趋势：波动较大，不稳定
- 问题识别：第3周完成率下降，原因是期中考试
- 应对策略：考试周可以降低目标（如每天10分钟即可）

**关键问题**：
- 哪些天漏打卡？原因是什么？
  - 作业多？ → 可以调整运动时间（早上或睡前）
  - 天气不好？ → 可以改室内运动（跳绳、拉伸）
  - 忘了？ → 需要设置提醒（闹钟或家长监督）
  
- 完成率下降的真实原因？
  - 目标太高？ → 降低目标（如300个→200个）
  - 缺乏动力？ → 增加游戏化元素（如挑战赛）
  - 身体不适？ → 暂停或降低强度

2. **营养改善情况**
   - 牛奶：保持/改善/退步
   - 碳酸饮料：减少/增加/持平
   - 零食：改善/退步/持平

3. **睡眠达标率**
   - 本周：__/7天达标
   - 上周：__/7天达标
   - 趋势：上升/下降/持平
   - 问题：哪些天超时？原因？

4. **情绪稳定性**
   - 整体心情：__分（1-5分）
   - 身高焦虑：__分（1-5分）
   - 学业压力：__分（1-5分）
   - 趋势：改善/恶化/持平

**其他维度数据**：

1. **学业成绩**
   - 错题攻克率：__%
   - 知识点掌握情况
   - 学习方法应用情况

2. **AI素养**
   - AI工具使用次数：__次/周
   - 新工具掌握情况
   - 应用场景扩展情况

3. **科创项目**
   - 项目产物数：__个/周
   - 项目推进速度
   - 遇到的问题

4. **表达能力**
   - 讲解流畅度：逐步提升/持平/下降
   - 当前阶段：对AI讲/对老师讲/录制视频/学校展示/比赛答辩

### 月度复盘重点

**身体发育**：

- 身高变化（测量值 vs 目标）
- 运动完成率趋势（4周数据）
- 睡眠达标率趋势（4周数据）
- 营养改善情况（4周数据）
- 情绪稳定性（4周数据）

**其他维度**：

- 学业成绩变化（班级排名变化）
- AI工具熟练度（能熟练使用的工具数）
- 项目推进速度（项目完成度）
- 表达能力提升（从哪个阶段到哪个阶段）

## 四、如何调整目标

### 调整原则

1. **数据驱动**：基于数据反馈调整，不要凭感觉
2. **渐进式**：不要一次性调整太多，每次调整1-2个目标
3. **可量化**：所有调整都要有明确的量化指标
4. **及时调整**：发现问题及时调整，不要等到月底

### 调整场景

**场景1：目标过高，孩子完不成**

**判断标准**：
- 连续2周完成率<70%
- 孩子明显有压力或抗拒

**调整方法**：
- 降低目标（如从85%降到75%）
- 简化执行方式
- 增加正向激励

**示例**：
```
原目标：运动完成率85%（6/7天）
调整后：运动完成率75%（5/7天）
原因：连续2周只完成71%，孩子有压力
```

**场景2：目标过低，孩子轻松完成**

**判断标准**：
- 连续2周完成率>95%
- 孩子觉得太简单

**调整方法**：
- 提高目标（如从75%提高到85%）
- 增加难度（如增加运动量）
- 增加新项目

**示例**：
```
原目标：每天跳绳300个
调整后：每天跳绳400个
原因：连续2周轻松完成，可以增加难度
```

**场景3：目标不合适，需要改变方向**

**判断标准**：
- 孩子不感兴趣
- 效果不明显
- 有其他更重要的目标

**调整方法**：
- 改变目标内容（如从跳绳改为篮球）
- 改变执行方式（如从每天改为每周3次）
- 暂停当前目标，启动新目标

**示例**：
```
原目标：每天跳绳300个
调整后：每周3次篮球，每次30分钟
原因：孩子对跳绳不感兴趣，更喜欢篮球
```

### 调整流程

1. **分析数据**：查看最近2-4周的数据
2. **识别问题**：找出需要调整的目标
3. **制定新目标**：设定新的量化指标
4. **更新话术**：给老师提供新的话术模板
5. **通知老师**：明确告知调整内容和原因
6. **跟踪效果**：1-2周后评估调整效果

## 五、如何优化流程

### 流程优化原则

1. **基于数据**：根据执行数据优化，不要凭感觉
2. **小步迭代**：每次优化1-2个环节，不要大改
3. **保持稳定**：核心流程保持稳定，只优化细节
4. **及时反馈**：优化后及时收集反馈，评估效果

### 优化场景

**场景1：时间分配不合理**

**判断标准**：
- 某个环节总是超时
- 某个环节总是提前结束
- 总时长经常超过90分钟

**优化方法**：
- 调整时间分配（如从20分钟改为15分钟）
- 简化环节内容
- 优化执行方式

**示例**：
```
原流程：实践练习20分钟
优化后：实践练习15分钟，快速复盘5分钟
原因：实践练习经常超时，复盘时间不够
```

**场景2：环节衔接不顺畅**

**判断标准**：
- 老师反馈衔接生硬
- 孩子反馈不自然
- 执行效率低

**优化方法**：
- 优化话术模板
- 增加过渡环节
- 调整环节顺序

**示例**：
```
原话术："舒尔特表做得不错。现在我们来看看身体发育。"
优化后："舒尔特表做得不错，注意力很集中。现在我们来看看身体发育的进度——先看看你这周的运动打卡记录。"
原因：过渡更自然，孩子更容易接受
```

**场景3：环节效果不明显**

**判断标准**：
- 数据没有改善
- 孩子不感兴趣
- 老师反馈效果差

**优化方法**：
- 改变环节内容
- 改变执行方式
- 增加互动性

**示例**：
```
原方式：营养环节只是询问和记录
优化后：营养环节增加游戏化元素（如"营养打卡表"）
原因：孩子不感兴趣，增加互动性
```

### 优化流程

1. **收集反馈**：从老师、孩子、数据三个维度收集反馈
2. **分析问题**：找出需要优化的环节
3. **制定方案**：设计优化方案
4. **更新文档**：更新执行手册和决策手册
5. **通知老师**：明确告知优化内容和原因
6. **跟踪效果**：1-2周后评估优化效果

## 六、如何监督执行

### 监督方式

1. **数据监督**：每周查看老师提交的记录
2. **现场监督**：偶尔旁听课程，观察执行情况
3. **反馈监督**：定期询问孩子和老师的反馈

### 监督重点

**执行准确性**：
- 是否严格按照流程执行
- 是否使用规定的话术
- 是否记录完整的数据

**执行质量**：
- 话术是否自然流畅
- 互动是否有效
- 孩子是否配合

**问题识别**：
- 是否有执行偏差
- 是否有问题未反馈
- 是否需要调整

### 监督流程

1. **每周检查**：查看老师提交的记录
2. **每月复盘**：分析月度数据，评估执行效果
3. **及时沟通**：发现问题及时与老师沟通
4. **持续优化**：根据监督结果优化流程和目标

## 七、常见决策场景

### 场景1：老师反馈"时间不够用"

**决策流程**：
1. 分析：哪个环节超时？为什么？
2. 判断：是执行问题还是流程问题？
3. 决策：
   - 如果是执行问题：要求老师提高效率
   - 如果是流程问题：调整时间分配或简化内容

**示例决策**：
```
问题：实践练习总是超时，导致总时长超过90分钟
分析：实践练习20分钟不够，但其他环节不能压缩
决策：将实践练习调整为15分钟，快速复盘调整为5分钟
```

### 场景2：孩子反馈"不感兴趣"

**决策流程**：
1. 分析：哪个环节不感兴趣？为什么？
2. 判断：是内容问题还是方式问题？
3. 决策：
   - 如果是内容问题：调整内容（如从跳绳改为篮球）
   - 如果是方式问题：优化执行方式（如增加游戏化元素）

**示例决策**：
```
问题：孩子对营养环节不感兴趣
分析：只是询问和记录，缺乏互动
决策：增加"营养打卡表"游戏化元素
```

### 场景3：数据没有改善

**决策流程**：
1. 分析：哪个维度数据没有改善？为什么？
2. 判断：是目标问题还是执行问题？
3. 决策：
   - 如果是目标问题：调整目标或改变方向
   - 如果是执行问题：优化流程或加强监督

**示例决策**：
```
问题：运动完成率连续4周都是71%，没有提升
分析：目标85%可能过高，孩子有压力
决策：将目标调整为75%，并增加正向激励
```

### 场景4：老师执行有偏差

**决策流程**：
1. 分析：哪些地方有偏差？为什么？
2. 判断：是理解问题还是态度问题？
3. 决策：
   - 如果是理解问题：重新说明流程和要求
   - 如果是态度问题：沟通或考虑换人

**示例决策**：
```
问题：老师经常跳过营养环节，只做运动
分析：可能是觉得营养环节不重要，或者时间不够
决策：明确强调四维一体缺一不可，并调整时间分配
```

- [ ] 分析数据趋势（上升/下降/持平）
- [ ] 识别需要调整的目标
- [ ] 制定本周目标和话术模板
- [ ] 通知老师本周安排

### 每周结束后

- [ ] 查看老师提交的记录

## 十、重要原则总结

### 1. 决策权在你，执行权在老师

- ✅ 你规定所有内容、流程、标准
- ✅ 老师只需要严格执行
- ❌ 不要让老师做决策
- ❌ 不要模糊边界

### 2. 数据驱动，不要凭感觉

- ✅ 基于数据做决策
- ✅ 量化所有目标
- ❌ 不要凭感觉调整
- ❌ 不要频繁调整

### 3. 渐进式调整，不要大改

- ✅ 每次调整1-2个目标
- ✅ 小步迭代优化
- ❌ 不要一次性大改
- ❌ 不要频繁调整

### 4. 及时沟通，保持透明

- ✅ 明确告知老师调整内容和原因
- ✅ 及时反馈问题和建议
- ❌ 不要突然改变
- ❌ 不要隐瞒问题
