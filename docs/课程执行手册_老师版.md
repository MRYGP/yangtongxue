# 课程执行手册（老师版）

## 使用说明

本手册是给**家教老师**使用的执行指南。你的任务是**严格按照本手册执行**，不需要做决策，只需要：

1. ✅ 按照流程执行
2. ✅ 记录数据
3. ✅ 反馈问题

**重要原则**：
- 所有内容、流程、标准都由家长规定
- 你只需要执行，不需要创新
- 遇到问题及时反馈给家长

---

## 一、90分钟课程标准流程

### 时间分配（必须严格遵守）

```
00:00-00:05  认知热身（舒尔特表）- 5分钟
00:05-00:20  身体发育（四维一体）- 15分钟
00:20-00:50  主题内容（学业/科创/AI/表达）- 30分钟
00:50-01:10  实践练习（动手做）- 20分钟
01:10-01:12  快速复盘（今天学到什么）- 2分钟
01:12-01:15  下次预告 - 3分钟
01:15-01:30  冥想时间（父子共修）- 15分钟
```

**重要**：总时长90分钟，不要超时，也不要提前结束。

---

## 二、环节1：认知热身（舒尔特表）- 5分钟

### 执行步骤

**准备阶段（1分钟）**：
1. 打开舒尔特表工具（APP或打印版）
2. 让孩子舒适就座
3. 简单说明："从1开始，按顺序找到并说出数字"

**训练阶段（3分钟）**：
1. 开始计时
2. 孩子从数字1开始，顺序找到并读出所有数字
3. 你在一旁观察，不给予提示
4. 如果卡住超过5秒，可以给一个轻微提示："往中间看"

**总结阶段（1分钟）**：
1. 记录完成时间（精确到秒）
2. 简单表扬："不错的速度！"（无论快慢都鼓励）
3. 询问感受："今天感觉怎么样？"

### 记录模板

```
舒尔特表：
- 时间：__秒（↓__秒 vs 上次）
- 状态：顺利完成/有困难（卡在数字__）
- 注意力：集中（1-5分）：__分
- 情绪：平和/烦躁
```

### 话术模板

**开场**：
"好的，我们先做5分钟的舒尔特表，让大脑开机。"

**结束**：
"舒尔特表做得不错，注意力很集中。现在我们来看看身体发育的进度——先看看你这周的运动打卡记录。"

## 三、环节2：身体发育（四维一体）- 15分钟

### 时间分配

- 运动：7分钟
- 营养：3分钟
- 睡眠：2分钟
- 心理：3分钟

### 维度1：运动（7分钟）

**A. 数据回顾（2分钟）**

查看家长提供的运动打卡数据，向孩子展示：

"我们先看看这周的运动打卡：
- 周一：跳绳__个，用时__分钟 ✅/❌
- 周二：...
- ...

本周完成率：__%（__/7天）
平均每次时长：__分钟"

**B. 趋势分析（2分钟）**

"我注意到：
- 进步：...
- 问题：...
- 亮点：..."

**C. 本周调整（3分钟）**

根据家长设定的目标，向孩子说明：

"本周目标：
- 完成率提升到__%（__/7天）
- 每次运动时长≥__分钟
- 新增项目：..."

**记录模板**：
```
【运动】（7分钟）
- 本周完成率：__%（__/7天）
- 进步：________________
- 问题：________________
- 下周目标：________________
```

### 维度2：营养（3分钟）

**第一次课（摸底）**：

"我们来看看你的营养情况：
- 你每天喝牛奶吗？多少毫升？
- 你每天吃鸡蛋吗？几个？
- 你喜欢吃鱼肉吗？每周几次？
- 你喝碳酸饮料吗？每周几次？
- 你吃零食多吗？什么类型的零食？"

记录后，根据家长提供的目标，给出建议。

**非第一次课（快速检查）**：

"这周营养方面：
- 牛奶：每天__ml ✅
- 鸡蛋：每天__个 ✅
- 碳酸饮料：__次 ✅/⚠️
- 零食：________________ ✅/⚠️

保持下去，下周同样的标准。"

**记录模板**：
```
【营养】（3分钟）
- 牛奶鸡蛋：________________
- 碳酸饮料：__次
- 零食：____________________
- 下周：____________________
```

### 维度3：睡眠（2分钟）

**第一次课（摸底）**：

"我们来看看你的睡眠情况：
- 你一般几点睡觉？几点起床？
- 你睡够9-10小时吗？
- 你睡前会玩手机或看电视吗？
- 你入睡快吗？大概多久能睡着？"

**非第一次课（快速检查）**：

"这周睡眠：
- 10点前入睡：__/7天 ✅
- 问题：________________
- 下周目标：__/7天达标"

**记录模板**：
```
【睡眠】（2分钟）
- 10点前入睡：__/7天
- 问题：____________________
- 下周目标：________________
```

### 维度4：心理/情绪（3分钟）

**A. 情绪观察（1分钟）**

"我们来聊聊这周的心情：
- 这周有什么开心的事吗？
- 有什么不开心或压力大的事吗？
- 你对自己的身高有压力吗？会因为这个不开心吗？
- 你觉得爸妈对你的期待压力大吗？"

**评分（1-5分）**：
```
情绪评分：
- 整体心情：__分
- 身高焦虑：__分
- 学业压力：__分
- 同伴关系：__分
```

**B. 心理建设（1分钟）**

根据评分，使用家长提供的话术模板：

| 情况 | 话术 |
|------|------|
| 身高焦虑重（1-2分） | "身高是可以努力的，我们一起做运动+营养+睡眠，一定会有效果。重要的是健康和自信，不是跟别人比。" |
| 学业压力大（1-2分） | "成绩5-10名已经很好了，不用太焦虑。我们用AI帮助学习，提高效率，省出时间做其他事。" |
| 同伴关系差（1-2分） | "有什么同学关系的问题吗？我们一起想想怎么改善。" |
| 整体状态好（4-5分） | "看起来心情不错，保持这种状态！" |

**C. 正向激励（1分钟）**

"我发现你这周：
- 运动坚持得很好，... ✅
- 营养也在改善，... ✅
- 睡眠还需要再努力一下，... ⏸️

你的努力我都看到了，继续加油！身高一定会有变化的。"

**记录模板**：
```
【心理】（3分钟）
- 情绪评分：__分
- 身高焦虑：__分
- 正向激励：________________
```

## 四、环节3：主题内容（30分钟）

### 轮换机制（由家长规定）

| 周几 | 主题 | 你的任务 |
|------|------|----------|
| 周一 | 学业辅导 | 错题分析+知识点讲解+实践练习 |
| 周三 | AI素养 | AI工具教学+实操练习+复盘总结 |
| 周五 | 科创项目 | 项目进度回顾+任务推进+下周计划 |
| 周日 | 表达训练 | 讲解本周内容+录制视频+观看回放 |

### 主题1：学业辅导（周一）

**30分钟流程**：
- 00:00-00:10  错题分析（AI辅助）
- 00:10-00:20  知识点讲解（三色笔法）
- 00:20-00:30  实践练习（当场做题）

**关键方法**：
- **AI辅助学习**：遇到不会的题先问AI，理解思路后自己做
- **三色笔法**：红色（不会）→ 蓝色（理解）→ 绿色（会讲）
- **闭书回忆**：讲解完后，闭上书复述知识点

### 主题2：AI素养（周三）

**30分钟流程**：
- 00:00-00:10  AI工具教学（新工具/新用法）
- 00:10-00:20  实操练习（用AI完成任务）
- 00:20-00:30  复盘总结（AI帮了什么/没帮什么）

**示例任务**（由家长提供）：
- 用AI生成思维导图
- 用AI辅助写作文
- 用AI生成编程思路
- 用AI分析错题

### 主题3：科创项目（周五）

**30分钟流程**：
- 00:00-00:10  项目进度回顾
- 00:10-00:20  本周任务推进（编程/设计/测试）
- 00:20-00:30  下周计划（拆解任务）

**关键原则**：
- 孩子主导，你辅助
- 小步快跑，每周至少推进1个产物
- 全程记录，方便后续复盘

### 主题4：表达训练（周日）

**30分钟流程**：
- 00:00-00:10  讲解本周学到的内容（对你讲）
- 00:10-00:20  录制视频（可重录）
- 00:20-00:30  观看回放+改进（找1个改进点）

**渐进式阶梯**（由家长决定当前阶段）：
1. 对AI讲（零压力）
2. 对老师讲（低压力）
3. 录制视频（可反复录）
4. 学校展示（中等压力）
5. 比赛答辩（高压力）

## 五、环节4：实践练习（20分钟）

**你的任务**：
- 让孩子动手做（做题/编程/设计/写作等）
- 观察孩子的操作过程
- 记录遇到的问题和亮点

**记录要点**：
- 完成情况：完成/未完成/部分完成
- 遇到的问题：具体描述
- 亮点：做得好的地方

## 六、环节5：快速复盘（2分钟）

**话术模板**：

"今天课程结束了，我们来快速回顾一下：
- 今天学到了什么？
- 哪个环节最有收获？
- 有什么困难吗？"

**记录要点**：
- 孩子说出的3个要点
- 情绪状态（积极/一般/消极）

"下次课我们继续：

- 身体发育：继续关注...（运动/营养/睡眠/心理）

## 八、环节7：冥想时间（15分钟）

### 准备阶段（2分钟）

**环境准备**：
- 关闭电脑/手机（或静音）
- 调暗灯光（如果可以）
- 坐姿：舒适即可（不必盘腿）

**开场白**：

"今天课程结束了，我们一起来放松一下。接下来15分钟，我们什么都不做，只是静静地坐着，感受呼吸。准备好了吗？"

### 冥想阶段（10分钟）

**引导词**（使用家长提供的录音，或按照以下模板轻声说）：

**00:00-02:00 | 身体扫描**

"闭上眼睛...感受你的身体坐在椅子上...感受你的脚接触地面...感受你的双手放在腿上...全身放松..."

**02:00-05:00 | 呼吸观察**

"现在把注意力放在呼吸上...感受空气从鼻子进入...感受胸腔的起伏...不要改变呼吸，只是观察它..."

**05:00-08:00 | 念头观察**

"你可能会有各种念头冒出来...没关系，不要抗拒，也不要跟随...就像看云飘过天空...念头来了，注意到它，然后让它飘走...再回到呼吸上..."

**08:00-10:00 | 感恩时刻**

"想一想今天你做得好的地方...可能是坚持运动，可能是解决了一道难题...在心里对自己说：我很棒，我在努力成长..."

### 结束阶段（3分钟）

**缓慢唤醒**：

"现在慢慢地，感受你的呼吸变得更深一点...慢慢地动动手指...动动脚趾...准备好的话，可以睁开眼睛..."

**分享时刻**：

- 你："你刚才有什么感受？"
- 孩子："..."（允许不说话，只是微笑）
- 你："我感觉很放松，和你一起冥想很好。"

**仪式结束**：

- 击掌或拥抱
- 宣布"下课了，现在是爸爸时间"

**记录模板**（可选）：
```
冥想记录：
- 时长：15分钟
- 孩子状态：安静坐完/中途动了/坐不住
- 情绪反馈：放松（1-5分）：__分
- 备注：________________
```

## 十、常见问题与应对

### Q1：孩子坐不住，冥想15分钟太长

**应对**：
- 第一次可以缩短到5分钟
- 逐步增加到10分钟、15分钟
- 不强求，鼓励为主
- **记录**：在课后记录中说明情况，让家长决定是否调整

### Q2：孩子问"为什么要做舒尔特表"

**话术**：
"这是训练大脑的方法，就像运动是训练身体。大脑也需要休息和训练。"

### Q3：运动打卡总是漏，怎么办

**应对**：
- 不要批评，分析原因（作业多？忘记？）
- 在记录中说明情况
- **不要自行调整目标**，让家长决定

### Q4：营养、睡眠、心理第一次课问不完

**应对**：
- 可以分2-3次课完成摸底
- 先做最重要的（运动），其他逐步完善
- 在记录中说明进度

### Q5：主题内容孩子不感兴趣

**应对**：
- 先完成规定流程
- 在记录中说明情况
- **不要自行改变主题**，让家长决定

### Q6：时间不够用，90分钟完不成

**应对**：
- 优先保证身体发育15分钟（不可压缩）
- 其他环节可以适当压缩，但要在记录中说明
- 让家长决定是否调整时间分配

## 十一、重要原则

### 1. 执行优先，不要创新

- ✅ 严格按照流程执行
- ✅ 使用提供的话术模板
- ❌ 不要自行改变内容
- ❌ 不要自行调整目标

### 2. 数据记录必须完整

- ✅ 每次课后立即记录
- ✅ 数据要准确、具体
- ❌ 不要遗漏任何环节

### 3. 遇到问题及时反馈

- ✅ 在记录中说明问题
- ✅ 向家长反馈异常情况
- ❌ 不要自行处理重大问题

### 4. 微调权限（5%以内）

- ✅ 可以在5%范围内微调（如时间分配）
- ✅ 紧急情况可临时调整，但需事后说明
- ❌ 超过5%需要家长同意
