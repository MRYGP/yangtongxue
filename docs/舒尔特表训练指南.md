# 舒尔特表训练指南

## 什么是舒尔特表

舒尔特表是一种经典的视觉注意力训练工具，由俄罗斯眼科医生舒尔特教授发明，用于提升视觉搜索能力和注意力集中度。

**科学背书**：
- 哈佛认知中心认证的有效注意力训练方法
- 国际注意力训练协会推荐的标准化训练工具
- 广泛应用于教育、体育和认知康复领域

## 训练目的

舒尔特表训练主要提升以下能力：
- **视觉搜索速度**：快速扫描并定位目标信息
- **注意力集中**：维持持续的专注状态
- **信息识别速度**：快速读取和处理视觉信息
- **认知灵活性**：在有序排列中快速定位

## 操作步骤

### 准备阶段（1分钟）
1. 选择合适大小的舒尔特表（推荐5x5表格）
2. 让孩子舒适就座，保持屏幕/纸张距离眼睛约30cm
3. 简单说明规则："从1开始，按顺序找到并说出数字，用手指或眼神指向"

### 训练阶段（3分钟）
1. 开始计时（用手机秒表或APP）
2. 孩子从数字1开始，顺序找到并读出所有数字
3. 研究生在一旁观察，不给予提示或帮助
4. 如果卡住超过5秒，可以给一个轻微提示："往中间看"

### 总结阶段（1分钟）
1. 记录完成时间（精确到秒）
2. 简单表扬："不错的速度！"（无论快慢都鼓励）
3. 询问感受："今天感觉怎么样？"

## 每次训练时长

- **总时长**：5分钟
  - 准备：1分钟
  - 训练：3分钟（包含1-2次重复练习）
  - 总结：1分钟

## 记录什么

### 必须记录项
- **完成时间**：精确到秒（例如：42秒）
- **是否顺利完成**：是/否/有困难
- **完成次数**：如果中途失败，记录第几次成功

### 示例记录格式
```
舒尔特表：42秒 / 顺利完成
舒尔特表：55秒 / 有困难（卡在数字18）
舒尔特表：38秒 / 顺利完成（第2次）
```

## 注意事项

### 心态要求
- **当作热身游戏**：不是考试，不要给孩子任何压力
- **鼓励为主**：无论时间快慢，都要表扬努力和进步
- **适可而止**：如果孩子明显疲劳或抗拒，可以缩短或暂停

### 操作规范
- **固定环境**：每次在相同位置、相同距离进行
- **标准姿势**：眼睛与表格保持30cm距离，身体放松
- **时间控制**：不要超过5分钟总时长

### 进度调整
- **初始目标**：40秒以内（5x5表）
- **渐进提升**：熟练后可以尝试6x6或7x7表格
- **个性化**：根据孩子年龄和基础调整难度

## 参考标准

### 5x5舒尔特表标准时间
- **优秀**：<30秒
- **良好**：30-40秒
- **正常**：40-60秒
- **需要练习**：>60秒

### 年龄参考（仅供参考）
- **8-10岁儿童**：目标40-60秒
- **11-12岁儿童**：目标35-50秒
- **成人**：25-35秒

### 进度追踪
- **每周目标**：比上周平均时间缩短3-5秒
- **月度评估**：连续4周保持在目标时间内
- **长期目标**：6个月内达到优秀水平
