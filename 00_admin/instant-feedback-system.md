# 即时反馈系统设计文档

**版本**：v1.0  
**更新日期**：2024-12-29  
**理论基础**：即时反馈回路理论（Instant Feedback Loop Theory）

---

## 一、理论基础

### 即时反馈回路核心公式

```
行为 → 即时反馈 → 多巴胺释放 → 想继续做 → 再行为
  ↑                                              ↓
  └──────────────── 正反馈循环 ─────────────────┘
```

### 五大核心要素

| 要素 | 教育版设计 | 为什么重要 |
|------|----------|----------|
| **1. 价值可量化** | 五维能力经验值 | 让孩子知道"这次课我进步了多少" |
| **2. 反馈即时性** | 课后10分钟内反馈 | 激活多巴胺回路 |
| **3. 进度可视化** | 三层进度（今日/本周/本月） | 产生"快要达成"的紧迫感 |
| **4. 动态调整** | 基于表现的加成+里程碑奖励 | 保持挑战性和新鲜感 |
| **5. 内在驱动** | 解锁权限+升级系统+家长认可 | 激活自主性动机 |

### 三层动机演化

```
外部奖励（经验值、解锁权限）
    ↓
内在满足（"我能做到"的成就感）
    ↓
身份认同（"我是一个会学习/会创造的人"）
```

---

## 二、系统设计

### 核心差异化（vs 美团骑手系统）

| 维度 | 美团骑手系统 | 教育版系统 | 设计原因 |
|------|------------|-----------|---------|
| **货币** | 真金白银 | 能力经验值+解锁权限 | 11岁孩子的真正"货币"是成就感和自主权 |
| **惩罚** | 超时扣钱 | ❌ 不扣分，只是不增加 | 避免焦虑，保护学习兴趣 |
| **反馈速度** | <1秒 | <10分钟（课后立即） | 教育场景下的最优平衡点 |
| **目标** | 短期（今日收入） | 短期+长期（今日+月度+学期） | 避免只追求短期指标 |
| **驱动力** | 外部→内在 | 内在为主，外部为辅 | 教育的长期价值 |

## 三、五维能力即时反馈系统

### 五维能力及经验值设计

#### 🥇 身体发育力（Primary，最高优先级）

**为什么是Primary**：窗口期不可逆，11岁是关键期

**经验值获取方式**：
- 完成1次运动（跳绳/拉伸）→ +8经验值
- 连续5天打卡 → +20经验值（里程碑）
- 10点前睡觉 → +5经验值
- 睡眠时长达标（9小时）→ +3经验值

**升级阈值**：
- 🌱 新手（0-20）
- 🌿 熟练（21-50）
- 🌳 专家（51-100）
- 🏆 大师（100+）

**解锁权限**：
- 熟练：可自选运动项目（跳绳/篮球/游泳）
- 专家：可设计自己的运动计划

---

#### 🥈 AI协作力（杠杆能力）

**为什么重要**：AI素养是杠杆能力，学会用AI，其他所有事都能做得更好

**经验值获取方式**：
- 独立使用AI解决问题 → +10经验值
- AI工具熟练度提升 → +50经验值（升级奖励）
- 用AI辅助项目设计 → +15经验值
- 教别人用AI → +20经验值

**升级阈值**：
- 🌱 新手（0-20）：需要帮助
- 🌿 熟练（21-50）：能独立使用
- 🌳 专家（51-100）：能创造性使用
- 🏆 大师（100+）：能教别人

**解锁权限**：
- 熟练：可用AI辅助学习（错题分析）
- 专家：可用AI辅助项目设计
- 大师：可教同学用AI

---

#### 🥉 学业掌握度

**定位**：基础要稳，维持班级前5-10名

**基础经验值获取方式**：
- 攻克1道错题 → +6经验值
- 知识点从表面→深入 → +10经验值
- 独立解题（不需要帮助）→ +8经验值
- 能讲清楚解题思路 → +5经验值

**高效学习方法加成**（基于科学研究）：
- **闭书回忆**（检索练习）→ 经验值×1.2倍
  - 科学证据：⭐⭐⭐⭐⭐ 非常强（检索练习比重复阅读效果好3.2倍）
  - 适用：概念性知识、结构化知识
  - 操作：合上课本，主动回忆，卡壳时先想3分钟再翻书
- **AI辅助检索练习** → 额外+8经验值
  - 用AI生成类似题目，闭书完成
  - 双重经验值：学业掌握度+8，AI协作力+10
- **睡前复习达标**（利用睡眠巩固记忆）→ 额外+5经验值
  - 科学证据：⭐⭐⭐⭐ 强（睡前30-60分钟效果最好）
  - 适用：需要记忆的内容（单词、古诗、基础知识）
  - 注意：只做"记忆类"任务，不做"思考类"任务
- **连续7天高效学习** → 额外+20经验值（里程碑）
  - 解锁权限：可自主设计学习计划

**升级阈值**：
- 🌱 新手（0-20）：需要帮助
- 🌿 熟练（21-50）：能独立完成
- 🌳 专家（51-100）：能讲清楚
- 🏆 大师（100+）：能教别人

**解锁权限**：
- 熟练：可自选学习内容
- 专家：可给同学讲题
- 大师：可设计自己的学习计划

**高效学习里程碑**：
- 连续3天使用闭书回忆 → 解锁"可自选复习内容"
- 连续7天完成高效学习流程 → 解锁"可自主设计学习计划"
- 闭书回忆准确率达90% → 解锁"可教同学学习方法"

---

#### 4️⃣ 创新思维力（Secondary）

**定位**：长期价值极高，孩子有兴趣

**经验值获取方式**：
- 项目推进1个关键步骤 → +10经验值
- 独立解决技术问题 → +15经验值
- 提出有价值的创新想法 → +8经验值
- 完成一个完整项目 → +50经验值（里程碑）

**升级阈值**：
- 🌱 新手（0-20）：跟着老师走
- 🌿 熟练（21-50）：能独立推进
- 🌳 专家（51-100）：能主导项目
- 🏆 大师（100+）：能设计整个项目

**解锁权限**：
- 熟练：可主导项目某个环节
- 专家：可自己设计项目
- 大师：可参加比赛

---

#### 5️⃣ 表达清晰度

**定位**：渐进式训练，不能急

**经验值获取方式**：
- 讲解流畅度提升10% → +7经验值
- 表达结构化（能用框架）→ +10经验值
- 录视频展示 → +20经验值（里程碑）
- 学校展示 → +30经验值（里程碑）

**升级阈值**：
- 🌱 新手（0-20）：磕磕巴巴
- 🌿 熟练（21-50）：能讲清楚
- 🌳 专家（51-100）：有框架
- 🏆 大师（100+）：能演讲

**解锁权限**：
- 熟练：可录视频展示
- 专家：可学校展示
- 大师：可参加比赛答辩

## 四、即时反馈机制

### 课后即时反馈卡（每次课后10分钟内）

**反馈速度**：<10分钟（符合即时反馈理论的"1-10秒=强效果"原则）

**反馈内容**：

```markdown
## 即时反馈卡（YYYY-MM-DD HH:MM）

### 🎯 本次课目标 vs 达成
- 目标：[具体目标]
- 达成：✅/⏸️ [完成度]

### 📊 五维能力变化（即时反馈）
┌──────────────────────────────────────┐
│ 🥇 身体发育力：+X经验值 [升级状态]   │
│    └─ [具体行为]                     │
│ 🥈 AI协作力：+X经验值 [升级状态]     │
│    └─ [具体行为]                     │
│ 🥉 学业掌握度：+X经验值 [升级状态]   │
│    └─ [具体行为]                     │
│ 4️⃣ 创新思维力：+X经验值 [升级状态]   │
│    └─ [具体行为]                     │
│ 5️⃣ 表达清晰度：+X经验值 [升级状态]   │
│    └─ [具体行为]                     │
└──────────────────────────────────────┘

### 🏆 本次解锁（立即可见的奖励）
- [解锁内容1]
- [解锁内容2]

### 📈 距离下一个里程碑
- [维度1]：再+X经验值 → [里程碑]（预计X次课）
- [维度2]：再[具体行为]X次 → [里程碑]

### 🎮 本周进度（短期可见）
- 今日：第X次课，完成度X%
- 本周：已完成X/Y次课，离本周目标还差X次

### 💪 下次小目标（30分钟可启动）
- [具体可执行的下一步1]
- [具体可执行的下一步2]
```

**多感官反馈设计**：
- **视觉**：经验值数字变化、升级特效（🎉）、进度条
- **听觉**：（可选）播放"升级成功"提示音
- **情感**：鼓励性语言（"太棒了！""升级了！"）

## 五、进度可视化

### 三层进度设计

#### 短期进度（今日/本次课）

**目的**：让孩子看到"立即可达成"的小目标

**设计**：
```
🎯 今日目标：完成3道错题
████████████████ 3/3（100%）
✅ 完成！解锁"自选题型"权限
```

**心理效应**："就差一点点了！再坚持一下！"

---

#### 中期进度（本周/本月）

**目的**：让孩子看到"阶段性成果"

**设计**：
```
🎯 本周目标：3次课全勤
██████████░░░░░░ 2/3（66%）
💡 再完成1次就解锁"周末自选学习内容"

🎯 本月目标：学业掌握度达到"专家"
████████░░░░░░░░ 38/50（76%）
💡 还差12经验值，预计2周后达成
```

**心理效应**："我已经做了这么多了，不能放弃！"

---

#### 长期进度（本学期）

**目的**：让孩子看到"长期积累"

**设计**：
```
🎯 学期目标：完成第一个科创项目
██████████████░░ 14/16步骤（87%）
💡 还差2步，本周可完成

🏆 累计成就
- 连续打卡：35天
- AI协作次数：47次
- 项目推进：14个里程碑
```

**心理效应**："我已经坚持这么久了，我是一个XXX的人！"

## 六、动态调整机制

### 四种奖励机制

#### 1. 基于表现的加成

```
- 连续3次课全勤 → 经验值×1.5倍
- 错题全对 → 额外奖励+10经验值
- 独立解决难题 → 额外奖励+15经验值
```

**心理效应**："我表现越好，奖励越多！"

---

#### 2. 里程碑奖励

```
- 连续5天运动打卡 → 解锁"周末自选运动项目"
- AI协作力达到"专家" → 解锁"可用AI辅助项目设计"
- 表达流畅度80% → 解锁"录视频展示"
- 完成第一个项目 → 解锁"可参加比赛"
```

**心理效应**："达成里程碑的成就感"

---

#### 3. 季节性活动

```
- 期中冲刺周：经验值翻倍
- 项目设计周：创新思维力经验值×2
- 表达训练周：表达清晰度经验值×2
```

**心理效应**："现在是特殊时期，多努力！"

---

#### 4. 意外惊喜（适度随机）

```
- 随机"超级任务"：完成可获得×3经验值
- 随机"挑战关卡"：完成可解锁特殊权限
- ⚠️ 频率控制：每月不超过2次，避免赌博成瘾
```

**心理效应**："下一次会不会有惊喜？"

## 七、内在驱动机制

### 自主性（Autonomy）：解锁权限系统

| 等级 | 经验值范围 | 解锁权限 |
|------|-----------|---------|
| 🌱 **新手** | 0-20 | 跟着老师走 |
| 🌿 **熟练** | 21-50 | 可自选部分学习内容、可自选运动项目 |
| 🌳 **专家** | 51-100 | 可主导下周计划某个环节、可用AI辅助项目设计、可给爸爸讲项目 |
| 🏆 **大师** | 100+ | 可设计自己的学习计划、可录视频展示、可参加比赛答辩 |

**设计原则**：
- ✅ 让孩子感觉"我在为自己做"
- ✅ 权限逐步解锁，避免一开始就有太多选择
- ✅ 高级权限需要真实能力支撑

---

### 胜任感（Competence）：升级系统

```
升级路径：
🌱 新手 → "我能做到基础任务"
    ↓
🌿 熟练 → "我能独立完成"
    ↓
🌳 专家 → "我能教别人"
    ↓
🏆 大师 → "我是这方面的高手"
```

**设计原则**：
- ✅ 新手任务简单，建立信心
- ✅ 难度递增，逐步提升挑战
- ✅ 即时反馈，让孩子看到进步

---

### 关联感（Relatedness）：社交认可

```
- 家长认可：每周展示雷达图，爸爸点评
- 社交分享：录视频可分享给同学
- 自我对比：与自己的历史记录对比（不是与同学竞争）
```

**设计原则**：
- ✅ 不与同学比，只与自己比（避免攀比焦虑）
- ✅ 家长认可是最重要的社交认可
- ✅ 分享成就是内在满足的外化

## 八、避免的五大陷阱

### 1. 过度依赖外部奖励 ❌

**错误做法**：用物质奖励解决一切

**正确做法**：
- ✅ 外部奖励（经验值、权限）用于启动行为
- ✅ 内在满足（成就感）用于维持行为
- ✅ 身份认同（"我是会学习的人"）用于长期坚持

---

### 2. 反馈延迟 ❌

**错误做法**：周末才统计本周表现

**正确做法**：
- ✅ 每次课后10分钟内反馈
- ✅ 每周日生成雷达图
- ✅ 每月生成成长报告

---

### 3. 目标不可见 ❌

**错误做法**：只有一个遥远的学期目标

**正确做法**：
- ✅ 三层进度：今日+本周+本月
- ✅ 里程碑节点清晰可见
- ✅ 实时显示进度

---

### 4. 奖励固定不变 ❌

**错误做法**：永远是"+10经验值"

**正确做法**：
- ✅ 基础奖励稳定
- ✅ 额外奖励动态（加成、里程碑）
- ✅ 适度随机性（意外惊喜）

---

### 5. 忽视内在动机 ❌

**错误做法**：只给经验值，不给自主权

**正确做法**：
- ✅ 解锁权限（自主性）
- ✅ 升级系统（胜任感）
- ✅ 家长认可（关联感）

## 九、技术实现

### 每次课后流程

```
1️⃣ 研究生记录4行
   （用时、做了什么、结果、下一步）
      ↓
2️⃣ 粘贴到GPTS
      ↓
3️⃣ GPTS自动生成"即时反馈卡"
   ├─ 计算五维能力经验值变化
   ├─ 判断是否升级
   ├─ 判断是否解锁新权限
   ├─ 计算距离下一个里程碑
   └─ 输出格式化反馈卡
      ↓
4️⃣ 发给孩子看（<10分钟）
      ↓
5️⃣ 多巴胺释放 🎉
      ↓
6️⃣ 想继续努力（内在驱动激活）
```

### 每周日流程

```
1️⃣ Claude Projects读取本周所有反馈卡
      ↓
2️⃣ 自动生成能力雷达图
      ↓
3️⃣ 生成本周成长报告
      ↓
4️⃣ 家长查看并点评
```

### 数据存储

```
03_logs/YYYY-MM-DD-feedback.md  ← 每次课的即时反馈
03_logs/YYYY-Www-ability-radar.md  ← 每周能力雷达图
00_admin/ability-levels.md  ← 五维能力当前等级
```

## 十、成功标准

### 短期成功标准（1个月内）

- [ ] 孩子能主动问"这次我能涨多少经验值？"
- [ ] 孩子能准确说出自己每个维度的当前等级
- [ ] 孩子能主动规划"下次我要做什么才能升级"

### 中期成功标准（3个月内）

- [ ] 至少2个维度达到"熟练"
- [ ] 孩子开始从"为了经验值"转变为"为了成就感"
- [ ] 孩子能自主选择学习内容和运动项目

### 长期成功标准（6个月内）

- [ ] 至少1个维度达到"专家"
- [ ] 孩子形成身份认同（"我是一个会学习/会创造的人"）
- [ ] 孩子主动分享成就（录视频、给同学讲）
