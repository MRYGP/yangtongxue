## 一票否决项（出现任意一项直接不录用）

- [ ] **试讲听不懂**：学生完全听不明白在讲什么
- [ ] **明确拒绝写记录**："我不写""口头说就行"
- [ ] **态度傲慢或不耐烦**：对学生不耐烦、批评、急躁
- [ ] **拒绝身体发育管理**："这不是我的职责""我只负责学习"
- [ ] **不愿意调整**："我会继续用我的方法""是孩子的问题"
- [ ] **边界严重不清**：迟到、随意请假、结算含糊
- [ ] **明确拒绝AI工具**："我不用这些东西""AI会让学生依赖"
- [ ] **定制研究能力极弱**：说不出诊断方法、方案笼统、无验证意识
