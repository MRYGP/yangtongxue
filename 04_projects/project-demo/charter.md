# Project Charter - Demo Project

project_slug: project-demo
start_date: 2025-12-20
status: 进行中

## Goal（目标）

验证"研究化 + 项目化"的家庭教育方法是否可行，建立可复制的协作流程。

## Success Criteria（成功标准）

- [ ] 建立可持续的运动打卡机制（完成率 ≥ 80%）
- [ ] 完成第一个完整项目周期（选题→设计→开发→展示）
- [ ] 形成可复制的协作流程文档
- [ ] 孩子能独立使用AI工具辅助学习

## Deliverables（产物清单）

1. 运动打卡机制文档
   - 包含打卡表、奖励机制、数据记录方法

2. AI工具使用指南
   - 包含ChatGPT使用场景、操作步骤、注意事项

3. 项目协作流程文档
   - 包含周计划模板、周复盘模板、项目文档模板

4. 周复盘模板
   - 包含数据摘要、洞察、调整、文件清单

5. 第一个完整项目案例
   - 包含项目选题、设计、开发、展示全流程

## Owner（责任人）

- Research Lead: 方案设计、复盘分析、关键沟通
- Ops Lead: 日常推进、打卡监督、资料整理
- 孩子: 项目主导、执行任务、记录过程

## Milestones（里程碑）

1. Week 1 (2025-W50): 建立基础机制
   - 状态: 已完成
   - 完成日期: 2025-12-22

2. Week 2 (2025-W51): 优化机制并开始项目设计
   - 状态: 已完成
   - 完成日期: 2025-12-22

3. Week 3-4 (2025-W52-2026-W01): 完成第一个完整项目周期
   - 状态: 进行中

## Risks（风险）

1. 风险: 孩子对打卡机制有抵触
   - 应对: 降低初始目标，先建立习惯

2. 风险: 项目周期过长，失去动力
   - 应对: 设置短期里程碑，及时反馈
