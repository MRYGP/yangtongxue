# Project Charter - 立体教育培养项目

## 目标

把孩子的成长过程，用**研究化 + 项目化**的方法推进，培养可迁移能力：学习能力、创造能力、表达能力、AI协作能力。

## 核心定位

### 这不只是为了小升初

小升初是第一个检验点，但真正目标是培养**长期受益的能力**：

| 能力         | 为什么重要               | 如何培养             |
| ------------ | ------------------------ | -------------------- |
| **发现问题** | 比技术更稀缺，决定做什么 | 训练观察、提问、分析 |
| **从0到1**   | 不是学知识，是做产品     | 完整走完项目全流程   |
| **表达展示** | 好东西要能讲清楚（孩子弱项，需重点训练） | 渐进式：对AI→对爸爸→录视频→学校展示→比赛答辩 |
| **迭代思维** | 第一版不完美很正常       | 测试→反馈→改进循环   |
| **AI素养**   | AI时代的基础能力         | 在项目各环节中自然习得使用AI工具 |

### 五个培养方向及优先级

| 优先级 | 方向 | 为什么这个优先级 | 执行方式 |
|-------|------|-----------------|---------|
| 🥇 | **身体发育（长高）** | 窗口期不可逆，11岁是关键期，错过无法弥补 | 每天运动+营养+睡眠监督 |
| 🥈 | **AI素养** | 杠杆能力，学会用AI，其他所有事都能做得更好 | 嵌入所有活动，不单独学 |
| 🥉 | **学业跟进** | 基础要稳，5-10名需维持，但不必all in | AI辅助高效学习 |
| 4 | **创新思维** | 长期价值极高，孩子有兴趣是天然优势 | 通过科创项目实践 |
| 5 | **表达能力** | 需要渐进，不能急，强迫可能适得其反 | 融入科创项目，逐级提升 |

## 短期与长期目标

**6个月目标**（2026年小升初前）：
- 建立科学的长高训练体系，养成运动习惯
- 完成第一个完整科创项目
- 学业稳定在前10名
- 熟练使用2-3种AI工具
- 能独立讲解项目5分钟

**长期目标**（初中及以后）：
- 身体健康发育，自信阳光
- 形成"发现问题→设计→实现→迭代"的能力闭环
- AI成为日常学习和创造的自然工具
- 每年完成1-2个有质量的项目
- 表达能力逐步提升，能清晰表达想法

## 里程碑

1. **Week 1-4**: 建立基础机制（运动打卡、AI工具使用）
2. **Week 5-8**: 完成第一个科创项目选题与设计
3. **Week 9-16**: 完成第一个完整科创项目（开发+展示）
4. **Week 17-24**: 优化机制，开始第二个项目

## 成功标准

- 运动打卡完成率 ≥ 80%
- 完成第一个完整项目周期（选题→设计→开发→展示）
- 形成可复制的协作流程文档
- 孩子能独立使用AI工具辅助学习
