# 招聘应聘者速览（减压版）

你应聘的不是只讲题的传统家教。
我们在做一个「AI + 人」的家庭教育协作项目：补课约30%，核心是把学习变成可复盘的小训练。

## 你需要做什么（每次课 3 步）

- **课前5分钟**：对齐“目标一句话”（本次课只抓一个最小主题 + 预期产出）
- **课中60–90分钟**：带孩子完成一小步，并留下**可验证证据**（一页解题过程/一段复述/一个小作品）
- **课后10分钟**：提交**4行事实记录**（用时/做了什么/结果/下一步一小步）

> 你只要写4行；“即时反馈卡”由 GPTS 自动生成并发给孩子看。模板：`docs/模板_课后即时反馈卡.md`

## 这份工作到底“难不难”

- 不难在“写作/长文”，我们反而禁止长文
- 难在“稳定”：稳定到场、稳定留证据、稳定按节奏复盘

## 工作量边界（先说清楚）

- **全周总投入**：约3–5小时（随实际上课频次略变）
- **备课**：每次不超过30分钟（工具会给框架）
- **记录**：每次不超过10分钟（只要4行事实）
- 默认脱敏：不写可识别信息

## 我们怎么验证匹配（先合作4周）

我们只验证一件事：你能否稳定跑通“上课 → 记录 → 调整”的闭环。

### 第1周：跑通一次闭环（最低成本）

- **一次15分钟试讲**：讲一个你最擅长的小点，让学生能复述一句话
- **一次完整上课**：课后提交1条4行记录 + 下一步小目标
- **一个小调整**：根据记录只改一个参数（题量/计时/口述步骤等）
- **一段小结（10句以内）**：有效的/无效的/下周保留与停止

验收标准很简单：记录里有证据；下一步清楚且30分钟能启动。

## 你能得到什么支持

- 课堂结构与记录模板直接给你
- 我每周陪你复盘一次，让你越做越省力
- 表现好会帮你把产出整理成可展示经历与作品（可脱敏）

补充：AI工具我们会教，你不用一开始就很会用。我们会给老师配 ChatGPT PLUS 账号，用 GPTS 帮你把“4行记录”自动变成反馈卡。


## 常见答疑（3条）

- **能教六年级吗**：可以。语文/数学/英语都可切入（孩子语文相对弱，数学基础不错）
- **记录会不会很重**：不重。每次课后10分钟，4行事实，不写作文
- **薪资怎么算**：按小时/按次结算更清楚（更适合兼职/在校生）。参考：75元/小时；每次到场通常1–2小时约150–180元/次；更长时段约200–250元/次（按强度）
