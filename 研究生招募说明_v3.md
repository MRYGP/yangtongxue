# 研究生招募：AI时代的立体家庭教育实验

## 一句话说明

我们正在寻找一位教育技术/教育学相关专业的研究生（硕士或博士），加入一个真实的家庭教育实验——**用AI工具辅助两个孩子的立体成长（身体发育、学业、创新思维、表达能力、AI素养）**。这既是一份家教工作，也是一个可以产出多篇论文的研究课题。

---

## 项目背景

**家庭情况**
- 地点：长沙市雨花区
- 孩子：两兄弟，11岁（六年级）和9岁（三年级）
- 家长：认知科学背景，熟悉AI应用，可提供研究方法论指导
- 资源：充足，硬件/材料/工具等投入不是问题

**我们相信**
- AI时代，教育方式需要革新
- 孩子需要的不只是补课，而是**立体的能力培养**
- AI工具可以让教育更科学、更个性化、更高效
- 这种模式值得研究、验证、推广

## 培养目标（五个方向，有明确优先级）

这是一个**立体的培养体系**，不只是补课：

| 优先级 | 方向 | 为什么这个优先级 | AI辅助方式 |
|-------|------|-----------------|-----------|
| 🥇 | **身体发育（长高）** | 11岁是窗口期，错过无法弥补 | AI制定运动计划、追踪数据、营养建议 |
| 🥈 | **AI素养** | 杠杆能力，学会用AI其他事都更高效 | 嵌入所有活动，不单独学 |
| 🥉 | **学业跟进** | 基础要稳，5-10名需维持 | AI辅助错题分析、知识点梳理、高效学习 |
| 4 | **创新思维** | 长期价值高，孩子有兴趣 | AI辅助选题、技术实现、项目文档 |
| 5 | **表达能力** | 需要渐进，融入其他活动 | AI模拟问答、演讲练习、录制复盘 |

**核心理念**：这五个方向是一个有机系统——

- **身体发育**是基础，保障精力和状态
- **AI素养**是工具层，嵌入所有活动，放大效率
- **学业、科创、日常**是应用场景
- **创新思维和表达能力**在实践中自然习得

## 工作方式（重要）

这不是传统家教，我们采用**研究型工作方式**：

### 核心流程

```
做方案 → 执行 → 记录 → 复盘 → 调整
```

### 具体要求

1. **全程使用AI工具辅助**
   - ChatGPT、Claude、Google AI Studio、Gemini等
   - 用AI做计划、分析问题、生成方案、复盘总结
   - 学习并掌握这些工具是工作的一部分

2. **文档化**
   - 每周提交工作文档到GitHub仓库
   - 包括：本周计划、执行记录、数据、复盘、下周调整
   - 使用Markdown格式

3. **与家长协作**
   - 家长会用AI审阅你的文档，给出反馈
   - 每周简短沟通，对齐方向
   - 共同优化方案

4. **研究视角**
   - 把这个项目当作一个教育实验来做
   - 记录过程数据、效果数据
   - 积累可用于论文的素材

## 应聘流程

### 第一步：提交申请

发送以下信息到 [邮箱/微信]：

1. 简单的自我介绍（专业、年级、研究方向）
2. 为什么对这个项目感兴趣
3. 你目前对AI工具的掌握程度（用过哪些、做过什么）
4. 你的时间安排是否匹配

### 第二步：完成试炼任务

我们会发给你一个小任务（约2小时），例如：

> 用任意AI工具，针对"帮助一个内向的11岁男孩准备科创项目答辩"设计一个训练方案。
> 要求体现AI工具的使用过程，以Markdown格式提交。

这个任务帮助我们了解：
- 你使用AI工具的真实能力
- 你的结构化思维和方案能力
- 你的文档表达能力

### 第三步：面谈

线下或线上详细沟通，双向了解。

### 第四步：一个月试用期

第一个月是磨合期：
- 了解两个孩子的情况
- 学习和熟悉各种AI工具
- 与家长对齐目标和方法
- 制定初步的培养计划

试用期结束后，双方评估是否继续长期合作。

A: 是的，这是工作方式的一部分。文档化有助于复盘和研究，也方便家长参与。

**Q: 可以先试一个月吗？**
A: 可以。第一个月是磨合期，双方都可以评估是否继续。
